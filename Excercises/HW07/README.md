## Purpose of Programs
__Q1__ - To calculate the average number of words per sentence from a paragraph in a .txt file.

__Q2__ - To accept as input a sentence in which all of the words are run together, but the first character of each word is uppercase. It then converts the sentence to a string in which the words are separated by spaces, and only the first word starts with an uppercase letter. 

__Q3__ - To do different functions with a file that cointains 654 sets of winning numbers from the Powerball Lottery.

__Q4__ - A Class that can be imported for later use.

__Q5__ - To create an object of a class and prompt the user to input data that will later be displayed on the screen.
